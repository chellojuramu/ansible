# Ansible Automation Repository

This repository is created to learn, practice, and maintain Ansible automation concepts from basics to real-world use cases.  
It contains Ansible playbooks, inventory files, and examples used for configuration management and server automation.

The goal of this repository is to build a strong foundation in Ansible for DevOps and Cloud roles.

---

## What is Ansible?

Ansible is an agentless automation tool used for:
- Configuration management
- Application deployment
- Server provisioning
- Infrastructure automation

Ansible uses YAML for writing playbooks and SSH for communicating with servers.
